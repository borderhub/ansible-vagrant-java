# ansible-vagrant-java


## Please prepare in advance

```$ brew install ansible```

```$ brew cask install vagrant```

```$ brew cask install virtualbox```

## clone

```git clone  https://github.com/shirabelass1001/ansible-vagrant-java.git```

## run

```$ vagrant up```
